## 🔧 部署和使用

### 安装要求
- Node.js 20.11.0+
- TypeScript 5.0+
- VS Code with MCP support

### 快速开始
```bash
# 1. 克隆项目
git clone <repository>

# 2. 安装依赖
cd cmmi-specs-agent/mcp-server
npm install

# 3. 构建项目
npm run build

# 4. 运行测试
npm test

# 5. 启动服务
npm start
```

### 配置文件
- **MCP配置：** `configs/mcp-config-insiders.json`
- **代理模板：** `agents/*.yaml`
- **类型定义：** `mcp-server/src/types/`
